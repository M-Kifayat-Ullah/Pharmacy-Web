# NJK Pharmacy Management System

A full-featured web-based pharmacy management system built with Python and Django.

## Screenshots

> Add screenshots here after uploading to GitHub

## Features

- Secure Admin Login & Logout
- Dashboard with real-time stats (Total Medicines, Low Stock, Total Invoices)
- Medicine Management (Add, Edit, Delete, Search)
- Stock Status tracking (Low Stock alerts)
- Billing / POS System with medicine search
- Invoice generation with automatic stock deduction
- Sales History with date, cashier, and total amount

## Technologies Used

- Python 3.13
- Django 5.2
- SQLite Database
- Bootstrap 5
- JavaScript (Fetch API / JSON)
- Font Awesome Icons

## Installation & Setup

1. Clone the repository
   
   git clone https://github.com/M-Kifayat-Ullah/Pharmacy-Web.git

2. Navigate to project folder
   
   cd Pharmacy-Web

3. Install dependencies
   
   pip install django

4. Run migrations
   
   python manage.py migrate

5. Create admin user
   
   python manage.py createsuperuser

6. Start the server
   
   python manage.py runserver

7. Open browser and go to
   
   http://127.0.0.1:8000
